8.Whatâ€™s the difference between window vs document?

Window Document
Definition Represents the browser window or tab where the document is loaded. Represents the HTML document loaded in the window.
Global Scope `window` object acts as the global object in the browser's JavaScript environment. `document` object is not the global object and is accessible within the scope of the current document.
Access Accessed using the `window` object prefix (e.g., `window.alert()`). Accessed directly without a prefix (e.g., `alert()`).
Properties Contains properties related to the browser window, such as `location`, `history`, `navigator`, etc. Contains properties related to the document structure and elements, such as `title`, `body`, `getElementById()`, etc.
Methods Provides methods for interacting with the browser window, such as `open()`, `close()`, `setTimeout()`, `setInterval()`, etc. Provides methods for manipulating and interacting with the document, such as `getElementById()`, `querySelector()`, `createElement()`, `addEventListener()`, etc.
Events Handles events related to the window or tab, such as `onload`, `onunload`, `onresize`, etc. Handles events related to the document and its elements, such as `onclick`, `onkeydown`, `onsubmit`, etc.
